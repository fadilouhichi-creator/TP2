# TP2 - Amelioration d'images

Projet realise pour le TP2 de **Computer Vision 1** avec **PyQt5** et **OpenCV**.

## Contenu du projet

- `design.ui` : interface Qt Designer
- `design.py` : code Python genere depuis `design.ui`
- `main.py` : logique complete de l'application
- `scripts/generate_report.py` : generation automatique du compte rendu
- `docs/Compte_rendu_TP2.md` : version Markdown du compte rendu
- `docs/Compte_rendu_TP2.pdf` : version PDF du compte rendu

## Fonctionnalites

- chargement d'une image en niveaux de gris
- affichage de l'image originale et de son histogramme
- egalisation d'histogramme
- seuillage binaire et seuillage d'Otsu
- filtres moyenneur, gaussien et median
- rotation, extraction et zoom numerique

## Lancement

```bash
python main.py
```

## Generation du compte rendu

```bash
python scripts/generate_report.py
```
